
Given:
True positive= TP=60
True negative=TN=40
False positive=FP=140
False negative=FN=9760


Find:

1) Accuracy

Accuracy=TN+TP/TN+FN+TP+FP

=40+60/40+9760+60+140
=100/10000
=0.01%

2) Recall

Recall=TPR=TP/TP+FN

=60/60+9760
=60/9820
=0.0006%

3) False Positive Rate

FPR=FP/FP+TN

=140/140+40
=140/180
=0.8%



4) True Negative Rate

TNR=TN/FP+TN

=40/140+40
=40/180
=0.2%

5) False Negative Rate

FNR=FN/FN+TP

=9760/9760+60
=9760/9820
=0.99%

6) Precision

Precision=TP/TP+FP

=60/60+140
=60/200
=0.3%

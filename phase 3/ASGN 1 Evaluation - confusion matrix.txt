
Given:
True positive= TP=60
True negative=TN=9760
False positive=FP=140
False negative=FN=40


Find:

1) Accuracy

Accuracy=TN+TP/TN+FN+TP+FP

=9760+60/9760+40+60+140
=0.982%


2) Recall

Recall=TPR=TP/TP+FN

=60/60+40
=0.6%


3) False Positive Rate

FPR=FP/FP+TN

=140/140+9760
=0.0141%




4) True Negative Rate

TNR=TN/FP+TN

=9760/140+9760
=0.985%


5) False Negative Rate

FNR=FN/FN+TP

=40/40+60
=0.4%


6) Precision

Precision=TP/TP+FP

=60/60+140
=0.3%
